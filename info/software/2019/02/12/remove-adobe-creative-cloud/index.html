    어도비 크리에이티브 클라우드 제거 방법

    어도비 크리에이티브 클라우드 데스크톱 언인스톨러에서 크리에이티브 클라우드를 시스템이 사용하고 있어 제거할 수 없다는 오류 발생시 해결하는 방법이다. 어도비 헬프 센터에서 아래의 파일을 내려받아 압축을 해제하고 실행하면 정상적으로 제거할 수 있다.

    • 윈도우즈 사용자
    • 맥OS 사용자

    파일 탐색기에서 크리에이티브 클라우드 바로 가기 제거 방법

    윈도우즈 파일 탐색기에 바로 가기가 남은 경우에는 아래의 방법을 따르면 된다.

    1. 키보드의 Windows + R 키를 눌러 실행 창을 열고 regedit를 입력하여 확인을 누른다.
    2. 레지스트리 편집기가 실행되면 키보드의 Ctrl(Control) + F 키를 누른다.
    3. 찾기 창이 뜨면 Creative Cloud Files를 찾을 내용에 입력하고, 찾을 대상에서 키와 값을 체크 해제한 후에 다음 찾기를 누른다.
    4. 아래 갈무리처럼 데이터 값에 Creative Cloud Files와 이름 값에 System.IsPinnedToNameSpaceTree가 있는지 확인한다. 만약 검색 결과가 다르다면 키보드의 F3 키를 눌러 비슷한 결과가 나올 때까지 검색한다.
    5. System.IsPinnedToNameSpaceTree를 더블 클릭하여 값 데이터를 0으로 변경한 후 확인을 누른다.
    6. 윈도우즈 파일 탐색기에서 바로 가기가 제거됐는지 확인한다.
