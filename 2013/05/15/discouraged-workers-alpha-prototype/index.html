Discouraged Workers - Alpha Prototype

Published May 15, 2013

I started working for Discouraged Workers using Ren’Py and Aram’s SCG Maker on April 21, 2013 and built a prototype on May 14, the same year. The prototype includes the current Demo volume.

After this I released version 0.1 on July 14th and continued it, finally I announced the complete of the project on March 12, 2018.

I haven’t build of this version because I lost it with the accident. I only have the installer, the files it used to build the installer, and the screenshots.
